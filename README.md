# Portfolio Website

A premium, minimal, and highly interactive developer portfolio built with Vite + React + Tailwind CSS + Framer Motion.

## Features

- **Modern Tech Stack**: React (Vite), Tailwind CSS, Framer Motion.
- **Premium Design**: Dark matte theme, glassmorphism, glowing accents.
- **Interactive**: Smooth scroll animations, hover effects, interactive terminal.
- **Responsive**: Fully optimized for all device sizes.
- **SEO Ready**: Semantic HTML structure.

## Structure

- `src/components/`: Reusable UI components (Navbar, Footer, Terminal, UI wrapper).
- `src/sections/`: Main page sections (Hero, About, Projects, etc.).
- `src/App.jsx`: Main application layout.
- `src/main.jsx`: Entry point.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Customization

- Update content in `src/sections/*.jsx`.
- Change colors in `tailwind.config.js` and `src/index.css`.
